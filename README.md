# 🚛 Delivery Management Bot (Logistics Engine)

Комплексная система автоматизации логистики и отчетности для служб доставки. Бот спроектирован для минимизации ошибок «человеческого фактора» при работе водителей и автоматизации бухгалтерии.

## 🛠 Стек технологий
* **Язык:** Python 3.10+
* **Фреймворк:** `aiogram 3.x` (Асинхронный бот)
* **База данных:** PostgreSQL (развернута на Railway)
* **ORM:** `SQLAlchemy` (Async)
* **Отчетность:** `pandas`, `openpyxl` (Excel), `FPDF` (PDF с поддержкой кириллицы)
* **Деплой:** Docker + Railway

---

## 🌟 Основные функции

### 👨‍✈️ Для водителей
* **Управление сменами:** Открытие/закрытие смены с фиксацией пробега и остатка топлива.
* **Система «Штурман» (Smart Route):** * Динамический список садиков с нумерацией и эмодзи-статусами (✅ — готово, 🚛 — текущий, ⏳ — в очереди).
    * **Защита от ошибок:** Система предупреждений «Off-route» при попытке отгрузки не по порядку.
    * Автоматическое обновление статусов в реальном времени.
* **Учет расходов:** Фиксация затрат на бензин и прочих расходов (обед, ремонт) при закрытии смены.

### 👑 Для администраторов
* **Конструктор маршрутов:** Привязка объектов к водителям и настройка очередности через админ-панель.
* **Управление БД:** Полный CRUD садиков, продуктов и пользователей.
* **Генерация отчетов:** Выгрузка детальных Excel и PDF отчетов за любой период с автоматическим расчетом чистой прибыли.

---

## 📊 Структура базы данных
* `Users`: Водители и администраторы (TG ID, роль, ФИО).
* `Kindergartens`: Объекты (Название, привязка к `driver_id`, номер в очереди `route_order`).
* `Shifts`: История смен (Пробег, топливо, расходы, выручка).
* `Deliveries`: Записи об отгрузках (Вес, тип продукта, сумма).

## 🚀 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/broccol1l/Logistics_Reporting_System
    cd Logistics_Reporting_System
    ```

2.  **Настройте переменные окружения (`.env`):**
    ```env
    BOT_TOKEN=your_telegram_bot_token
    DATABASE_URL=postgresql+asyncpg://user:pass@host:port/dbname
    ADMIN_ID=your_telegram_id
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Важно:** Для PDF-отчетов убедитесь, что файл `arial.ttf` находится в корне проекта.
